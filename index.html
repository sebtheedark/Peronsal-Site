The purpose of this project was to utilize various different analytical tools to be able to query data, and visualize charts on finding the locations that provided the most savings in terms of purchasing used cars. Data set was obtained from Kaggle and contained basic info on locations/dealers, pricing, date of upload and date of sale, thus showing a turnaround time for used car sales. Based on the prices, we were able to identify the most expensive cars and overpriced cars.

The data was uploaded to a hadoop apache cluster, once uploaded, the data was filtered and parsed through 66 different columns with thousands of rows, which was then put into tables via HQL/SQL to be able to extract the exact data that was needed to be able to visualize and answer our questions.
